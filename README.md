# Kyoto v2.0 - Local AI Factory

A self-hosted AI agent that routes tasks, learns preferences, executes code safely, and generates reports.

## Quick Start
1. cd src
2. python3 cli.py
3. Try: help, weather Tunis, stats

## Core Features
- Keyword routing & model preference learning
- Safe Python sandbox with timeout
- ComfyUI/FLUX image generation
- Live APIs: weather, crypto, news, web search
- PDF/text extraction & auto-file sorting
- Automated Markdown weekly reports

## Setup
pip3 install requests openai-whisper duckduckgo-search pypdf pandas reportlab Pillow
cd src && python3 cli.py
